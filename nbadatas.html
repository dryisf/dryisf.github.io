NBA Datas

projet scolaire

Création en groupe d'une application web à l'aide de l'API balldontlie affichant les statistiques des joueurs de NBA actifs durant la saison 2019-2020.

Pour ce projet, en ce qui concerne la partie back, je me suis chargé de récupérer les données renvoyées par l'API pour ensuite les stocker dans une base de données MongoDB. A l'aide du framework Express.js, j'ai conçu notre propre API puis ai intégré les données à la partie front en utilisant le framework Vue.js.
